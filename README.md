# Netflix Data Analysis 

## Project Overview

This project explores the Netflix dataset to understand trends in movies and TV shows available on the platform.
Using Python and data analysis libraries, the dataset is cleaned, analyzed, and visualized to uncover patterns such as content distribution, release trends, and popular genres.

The goal of this project is to practice data cleaning, exploratory data analysis (EDA), and data visualization using real-world data.

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
* Excel / Power BI (for additional visualization)

---

## Dataset

The dataset contains information about Netflix titles including:

 Title
 Type (Movie or TV Show)
 Director
 Cast
 Country
 Release Year
 Rating
 Duration
 Genre

---

## Project Steps

### 1. Data Loading

* Imported dataset using Pandas
* Examined structure and column information

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Converted data types where necessary

### 3. Exploratory Data Analysis

Performed analysis to answer questions such as:

 What is the distribution of Movies vs TV Shows?
 Which countries produce the most Netflix content?
 How has Netflix content grown over the years?
 What are the most common content ratings?

### 4. Data Visualization

Created visualizations including:

 Movies vs TV Shows distribution
 Content release trends by year
 Country-wise content distribution
 Rating distribution

---

## Key Insights

 Netflix contains more **Movies than TV Shows**.
 The majority of content was released after **2015**, showing rapid platform growth.
 The **United States and India** produce a large portion of Netflix content.
 Certain ratings such as **TV-MA and TV-14** appear most frequently.
